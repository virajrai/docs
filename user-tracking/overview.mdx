Layup's lightweight user identification system helps you deliver tailored experiences while maintaining simplicity and privacy. By connecting user interactions across your application, you can create seamless, personalized journeys without complex implementation.

## Key Benefits

- **Frictionless Recognition**: Identify returning users automatically, even before authentication
- **Contextual Experience**: Display relevant content based on organization or team membership
- **Behavior Insights**: Understand how different user segments interact with your product

## What data is tracked

Layup's user tracking system is designed to be simple and privacy-focused. We only track:

- **userId** - A unique identifier for each user
- **groupId** - An optional identifier for the organization or group the user belongs to

For implementation details, see the [Implementation](/user-tracking/implementation) guide.
