## Targeting Options

Announcements can be precisely targeted based on multiple factors:

<AccordionGroup>
  <Accordion title="User Attributes">
    Segment by user role, organization, subscription tier, or any custom attribute.
  </Accordion>
  <Accordion title="Behavioral Triggers">
    Show based on actions taken (or not taken), feature usage, or milestones achieved.
  </Accordion>
  <Accordion title="Application Context">
    Target by current page, feature being used, or position in a workflow.
  </Accordion>
  <Accordion title="Temporal Factors">
    Schedule by time of day, date range, days since signup, or session frequency.
  </Accordion>
</AccordionGroup>

## Configuration Options

Announcements can be customized through the [Layup Dashboard](https://app.layuplabs.ai/announcements):

- **Appearance**: Style announcements to match your brand and UI
- **Placement**: Control where announcements appear in your application
- **Dismissal Options**: Set whether users can dismiss, snooze, or must acknowledge
- **Frequency Caps**: Limit how many announcements a user sees in a given timeframe
