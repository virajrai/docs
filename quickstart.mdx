Get started with Layup in under 5 minutes.

## Prerequisites

- Node.js 16.0 or later

- A React-based application (React 18+)

- An account on [Layup Admin Dashboard](https://app.layuplabs.ai)

## Installation

<Steps>
  <Step title="Grab an API key from the Layup dashboard.">
    * Grab your API key from the [Layup dashboard](https://app.layuplabs.ai/settings/api-keys).
  </Step>

  <Step title="Install the Layup SDK using your preferred package manager.">
    <CodeGroup>
      ```bash npm
      npm install layup-js
      ```

      ```bash pnpm
      pnpm add layup-js
      ```

      ```bash yarn
      yarn add layup-js
      ```
    </CodeGroup>

  </Step>

  <Step title="Configure your environment variables.">
    Add the appropriate Layup API key into your `.env` file.

    ```bash
    NEXT_PUBLIC_LAYUP_AUTH_TOKEN=layup_*********************
    ```

  </Step>

  <Step title="Wrap your app with the LayupProvider.">
    In your root file, (e.g., `layout.tsx`, `index.tsx`, `_app.tsx`) wrap your app with the `LayupProvider`.

    ```typescript
    import { LayupProvider } from 'layup-js';

    function App() {
      return (
          <LayupProvider apiKey={process.env.NEXT_PUBLIC_LAYUP_AUTH_TOKEN!}>
            {children}
          </LayupProvider>
      );
    }
    ```

    <Tip>
      For optimal performance, place the `LayupProvider` as close as possible to the root of the application.
    </Tip>

  </Step>

  <Step title="Test the integration.">
    Once you run your app, Layup widget should now appear in the bottom-right corner.
  </Step>
</Steps>
